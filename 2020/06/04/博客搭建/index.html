安装git（windows版）

  1. 下载 git

  2. 生成SSH密钥 

    $ git config --global user.name "your github username"
$ git config --global user.email "your email signed up for github"
$ ssh-keygen -t rsa -C "your email signed up for github"

    密钥在 C:\Users\用户名\.ssh\id_rsa.pub 文件中   ‘用户名’ 替换成本机用户名

  3. 在Github中添加SSH

    进入Github的settings中的SSH and GPG keys粘贴密钥即可
